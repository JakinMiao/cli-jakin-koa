##cli-jakin-koa

基于koa的web 脚手架

####软件版本

- Node 10.16.3+
- npm 5+
- typescript 3.6+

#### 使用说明
Node 版本 v10 以上
- `npm start`- 启动
- `npm test` - 测试
- `npm run serve-ts` - 调试(修改后台自动重启)
- `npm run build-ts` - 编译
- `npm watch-ts` - 自动编译
- `npm tslint` - tslint格式检查

#### 打包说明

make build TAG=xxx 根据部署环境 构建不一样的包
