##cli-jakin-koa

基于koa的web 脚手架

####软件版本

- Node 10.16.3+
- npm 5+
- typescript 3.6+

#### 使用说明
Node 版本 v10 以上
- `npm start`- 启动
- `npm test` - 测试
- `npm run serve-ts` - 调试(修改后台自动重启)
- `npm run build-ts` - 编译
- `npm watch-ts` - 自动编译
- `npm tslint` - tslint格式检查

#### QuickStart
更新日期 2020-02-09

##### clone 本项目
```
git clone https://github.com/JakinMiao/cli-jakin-koa.git
```
#### 使用 ycli-koa
```
npm i ycli-koa -g
ycli-koa init projectname
```
